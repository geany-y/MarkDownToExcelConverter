# AntiGravity 包括的テスト用 Markdown

このファイルは、AntiGravity の変換機能をすべて網羅するためのテストデータです。

## 1. 見出しレベル (Headers)
# 見出し H1
## 見出し H2
### 見出し H3
#### 見出し H4
##### 見出し H5
###### 見出し H6

## 2. インライン書式 (Inline Formatting)
これは通常の段落です。
**このテキストは太字です（Bold）**
*このテキストは斜体です（Italic）*
~~このテキストは取り消し線付きです（Strike）~~
***これは太字かつ斜体です（Bold & Italic）***
`ここはインラインコードです`
※インラインコードは背景色ではなく、濃い赤色の文字（#A31515）で表現されます。

## 3. リストとインデント (Lists & Indentation)
### 3.1. 箇条書きリスト
- 項目 1
- 項目 2
    - ネストされた項目 2-1 (インデント 1)
        - ネストされた項目 2-1-1 (インデント 2)
- 項目 3
※箇条書きのハイフンは「・」に変換されて出力されます。

### 3.2. 番号付きリスト（連番自動修正テスト）
1. 手順 1
1. 手順 2 (Markdownでは1.だが、Excelでは2.になるはず)
    1. サブ手順 2-1
1. 手順 3 (Markdownでは1.だが、Excelでは3.になるはず)

## 4. ハイパーリンクと画像 (Links & Images)
### 4.1. 複数のハイパーリンク (同一行)
[Google](https://www.google.com) へのリンクと、[GitHub](https://github.com) へのリンクが同じ行にあります。
※Excel上ではテキスト内に [1], [2] が付与され、巻末のリンク一覧と対応するはずです。

### 4.2. 画像 (代替テキスト)
![AntiGravity ロゴ](https://example.com/logo.png)
※画像は代替テキストが出力され、背景色が適用されるはずです。

## 5. 特殊要素 (Special Elements)
### 5.1. 引用 (Blockquotes)
> これは引用です。
> 複数行にわたる引用もサポートされます。
> > ネストされた引用も可能です。

### 5.2. コードブロック (Code Blocks)
```typescript
function hello() {
  console.log("Hello, AntiGravity!");
}
```
※コードブロックは背景色はなく、文字色がダークブルー（#000080）で等幅フォントになります。

### 5.3. 水平線 (Horizontal Rule)
---
上の行は、ハイフン30個のテキスト「------------------------------」に変換されるはずです。

## 6. 除外対象 (Exclusions)
### 6.1. 表 (Tables)
| ヘッダー 1 | ヘッダー 2 |
| :--- | :--- |
| データ 1-1 | データ 1-2 |
| データ 2-1 | データ 2-2 |
※表は変換されず、プレーンテキストとして「表：| ... |」の形式で出力されるはずです。
